1. NumPy Broadcasting

When you do States * Gen[:, None], NumPy virtually stretches the 1D generator capacity list across all 87,600 columns instantly in C-memory, avoiding massive Python loops.

2. The CumSum Trick

np.cumsum is the secret to vectorized dispatch. It figures out exactly how much capacity is "ahead in line" for every hour simultaneously, so the system knows when the load is satisfied.

3. Boolean Masking

To count events, is_failing & ~is_failing_padded[:-1] compares an array against a shifted version of itself to find exact moments where 0 turns to 1 (A new failure event) across the whole decade.
