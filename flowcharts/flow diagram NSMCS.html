NSMCS Advantage

Fast and robust for island grids with many units. It treats every "snapshot" of the system as an independent event, allowing for extremely fast sampling without chronological tracking.

Computational Detail

The code uses np.random.random inside the loop to avoid the memory overhead of creating a massive matrix upfront, trading some vectorization speed for absolute memory safety.
