Sequential Monte Carlo Logic

Chronological Reliability Simulation for Power Systems

Step 1: Setup

Initialize Environment

Load Generator Data (Capacity, MTTF, MTTR) and the 8,760h Load Curve. Generate monthly hydro capacity caps lookup.

Step 2: Start State

Initial Sampling

Set all units to UP (State 0). For each unit, sample the first time-to-failure (TTF) using: T = -MTTF * ln(rand)

Loop Guard

Simulation Loop

Continue while
current_time < total_study_hours

Step 3: Supply/Demand

Capacity Assessment

  • Thermal: Sum capacities of all units in State 0.
  • Hydro: Sum UP units, then apply Monthly Cap.
  • Load: Fetch demand for current_time % 8760.
Step 4: Reliability Logic

Loss of Load Check

If Supply < Demand:
Increment total_LOL_hours by the time_step.

Step 5: Event Advancement

Dynamic Time Step

Advance clock by step = min(1.0, min_time_to_event).
Decrement all unit event timers by step.

Step 6: State Flip

Update Unit States

Identify units where timer ≤ 0.
0 → 1: Unit failed (sample TTR).
1 → 0: Unit repaired (sample TTF).

Repeat for N Years
